# Marriage Biodata – [Your Name]

A modern, dynamic marriage biodata website built as a stunning single-page portfolio — a unique alternative to traditional PDF biodatas common in Gujarati matrimonial circles.

## ✨ Concept

- Share a **PDF** with a **QR code** embedded → QR links to this website
- First glance: PDF for quick reference
- Next stage: Scan QR → visit this live, beautiful biodata page

## 🌐 Live Site

> [https://marriage-biodata.onrender.com](https://marriage-biodata.onrender.com)

## 📁 Project Structure

```
Marriage BioData/
├── index.html      # Full single-page biodata (HTML + CSS + JS)
├── profile.png     # Profile photo
├── render.yaml     # Render deployment config
└── README.md       # This file
```

## 🚀 Deployment

Hosted on **Render** (free static site).  
Auto-deploys on every push to `main` branch.

## 🛠 Local Preview

Just open `index.html` in any browser — no build step needed.

## 📄 Features

- Gujarati-inspired crimson & gold design
- Animated hero banner with glowing profile photo ring
- Auto age calculation from DOB
- Sections: Personal · Astrological · Family (Paternal + Maternal) · Education · Career · Hobbies · Partner Expectations · Contact
- WhatsApp / Call / Email CTAs
- Mobile-first, fully responsive
- Scroll-reveal animations
